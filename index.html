Киберполигон

Инновационная платформа для подготовки специалистов в области кибербезопасности.

Аннотация проекта

«Киберполигон» — это образовательная и исследовательская среда для тренировки навыков защиты и атаки в безопасном виртуальном окружении.

  • 75 участников
  • Куратор: Гневшев А.Ю.

О проекте

Введение

Платформа объединяет React-фронтенд, Django-бэкенд и виртуализацию (Docker, KVM, VirtualBox) для моделирования кибератак.

Проблемы

  • Ограниченная практика с реальными кейсами.
  • Устаревшие учебные методики.
  • Нехватка инфраструктуры для тестирования.
  • Разрыв между вузами и индустрией.
  • Дефицит квалифицированных специалистов.

Решение

Веб-платформа

React + Django с удобным интерфейсом и управлением виртуальными средами.

Симуляции

Интеграция Docker, KVM и VirtualBox для изолированных лабораторий.

Модули

CTF-задания, квесты и тренажёры.

Партнёры

Сети университетов и ИТ-компаний для практики и финансирования.

Участники

  • VMware, Vagrant, VirtualBox, Docker
  • Настройка Windows Server 2016

Описание вклада...

  • Vagrant, VirtualBox
  • Kali Linux, Metasploit
