Аннотация проекта

«Киберполигон» — это образовательная и исследовательская среда для тренировки навыков защиты и атаки в безопасном виртуальном окружении.

  • 75 участников
  • Куратор: Гневшев А.Ю.

О проекте

Проблемы

  • Дефицит практики с реальными кейсами.
  • Устаревшие методы обучения.
  • Нехватка инфраструктуры.
  • Разрыв между вузами и индустрией.
  • Дефицит квалифицированных кадров.

Решение

Веб-платформа

React + Django с удобным интерфейсом и управлением виртуальными средами.

Симуляции

Интеграция Docker, KVM и VirtualBox для изолированных лабораторий.

Модули

Учебно-тренировочные задания: квесты и CTF-соревнования.

Партнёрская сеть

Сотрудничество с ИТ-компаниями и университетами для практики и финансирования.

Участники

  • VMware, Vagrant, VirtualBox, Docker
  • Настройка Windows Server 2016

Изучил системы виртуализации, участвовал в развёртывании лабораторий.
